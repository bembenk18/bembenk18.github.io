SSH Ke Mesin Lokal Dengan Tailscale

2023-04-14
0 minute read
Linux , Server , Log

Tailscale adalah aplikasi opensource berbasis WireGuard, tailscale memungkinkan untuk membangun jaringan peer to peer private. Dalam case saya gunakan untuk akses Home server.

Installasi

curl -fsSL https://tailscale.com/install.sh | sh

Start Tailscale service

sudo tailscale up

Saat pertama kali akan diminta login, silahkan login dengan link yang muncul dikonsole, jika berhasil dijalankan kita akan mendapatkan ip seperti dibawah ini:

Dokumentasi

Kesimpulan

Dengan menggunakan Tailscale kita bisa membuat jaringan private yang mudah dan aman untuk akses server lokal dari internet.
