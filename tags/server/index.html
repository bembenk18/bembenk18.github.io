SSH Ke Mesin Lokal Dengan Tailscale

Tailscale adalah aplikasi opensource berbasis WireGuard, tailscale memungkinkan untuk membangun jaringan peer to peer private. Dalam case saya gunakan untuk akses Home server. Installasi curl -fsSL https://tailscale.com/install.sh | sh Start Tailscale service sudo tailscale up Saat pertama kali akan diminta login, silahkan login dengan link yang muncul dikonsole, jika berhasil dijalankan kita akan mendapatkan ip seperti dibawah ini: Web Interface Untuk mempermudah mengelola service Tailscale menyediakan web interface di link berikut Test akses SSH Test akses SSH ke server Dokumentasi Repo dan dokumentasi tentang Tailscale bisa diakses di https://github.

Home Server Armbian

Armbian adalah salah satu distro linux yang mendukung processor arm64 dengan spesifikasi rendah, salah satunya adalah STB HG680P yang banyak beredar dengan harga under 300k. STB ini juga bisa dijadikan alternative dari Raspberry Pi yang harganya semakin naik. Spesifikasi RAM : 2 GB ROM Internal : 8 GB CPU : Quad Core Amlogic s905x Cortex-A53 @ up to 1.51 GHz Dengan spesifikasi tersebut sudah cukup untuk menjalankan Armbian. Untuk Armbian saya pasang pada microSD Sandisk Ultra UHS-I 32GB.
